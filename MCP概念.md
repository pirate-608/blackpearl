# MCP（Model Context Protocol）概念

## 概述

**MCP**，全称 **Model Context Protocol（模型上下文协议）**，是由 **Anthropic** 于 2024 年 11 月推出的一个**开放标准和开源框架**。

## 核心定义

MCP 旨在标准化 AI 系统（如大型语言模型 LLM）与外部工具、系统和数据源之间的**集成与数据共享方式**。它为以下操作提供了统一的标准接口：

- 📖 **读取文件**
- ⚙️ **执行函数**
- 📝 **处理上下文提示（prompts）**

## 关键意义

- 在 MCP 出现之前，每个 AI 模型与外部工具的集成方式各不相同，缺乏统一标准。
- MCP 提供了一个**通用协议**，让不同的 AI 模型能够以一致的方式连接外部世界。
- 该协议发布后，已被 **OpenAI** 和 **Google DeepMind** 等主流 AI 厂商采纳。

## 简单类比

可以把 MCP 理解为 AI 世界的「USB-C 接口」——在它出现之前，每个设备和工具都需要各自的转接头；有了 MCP 之后，所有模型和工具都可以通过同一个标准接口互联互通。
