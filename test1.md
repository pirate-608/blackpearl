# 积分公式

## 1. 牛顿-莱布尼茨公式（微积分基本定理）
$$
\int_{a}^{b} f(x) dx = F(b) - F(a)
$$
其中 $F'(x) = f(x)$

## 2. 不定积分公式
- 幂函数：$\int x^n dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$
- 指数函数：$\int e^x dx = e^x + C$
- 对数函数：$\int \ln x dx = x\ln x - x + C$
- 三角函数：$\int \sin x dx = -\cos x + C$
- 三角函数：$\int \cos x dx = \sin x + C$

## 3. 基本积分项

| 函数类型 | 原函数 |
|---------|--------|
| $x^n$ | $\frac{x^{n+1}}{n+1}$ |
| $\frac{1}{x}$ | $\ln|x|$ |
| $e^{ax}$ | $\frac{e^{ax}}{a}$ |
| $\sin(ax)$ | $-\frac{\cos(ax)}{a}$ |
| $\cos(ax)$ | $\frac{\sin(ax)}{a}$ |
| $\tan(ax)$ | $-\frac{1}{a}\ln|\cos(ax)|$ |
| $\ln(x)$ | $x\ln(x) - x$ |
| $e^x$ | $e^x$ |
| $\sinh(x)$ | $\sinh(x)$ |
| $\cosh(x)$ | $\cosh(x)$ |
